# CLON DE NETFLIX

## DESCRIPCIÓN

El proyecto "Clon de Netflix" es una aplicación web que simula la funcionalidad de la popular plataforma de streaming Netflix. Construido utilizando HTML, CSS, JavaScript y Bootstrap, este proyecto permite a los usuarios explorar e interactuar con una colección de películas.

## CARACTERÍSTICAS

1. **Navegación y Búsqueda**:
   - La aplicación cuenta con un menú de navegación responsivo con enlaces para diferentes funcionalidades.
   - Los usuarios pueden buscar películas por título, año, calificación o fecha de estreno.

2. **Visualización de Películas**:
   - La aplicación muestra tarjetas de películas, cada una con el póster, título, año, fecha de estreno, género y calificación de la película.
   - Los usuarios pueden pasar el mouse sobre una tarjeta de película para ver un resumen de la trama.

3. **Ordenamiento y Filtrado**:
   - Los usuarios pueden ordenar las películas por calificación (top 3), fecha de estreno o título (alfabético).
   - La aplicación proporciona una funcionalidad de búsqueda para filtrar películas según varios criterios.

4. **Diseño Responsivo**:
   - La aplicación está diseñada para ser responsiva y optimizada para diferentes tamaños de pantalla, asegurando una experiencia de usuario fluida en múltiples dispositivos.

## TECNOLOGÍAS UTILIZADAS

- **HTML**: Utilizado para estructurar el contenido y el diseño de las páginas web.
- **CSS**: Utilizado para estilizar la aplicación y crear un diseño visual atractivo.
- **JavaScript**: Utilizado para agregar interactividad, manejar eventos de usuario y manipular el DOM.
- **Bootstrap**: Utilizado para facilitar el diseño responsivo y proporcionar componentes de interfaz de usuario pre-diseñados.

## COMENZANDO

1. Clona el repositorio:

   ```bash
   git clone https://github.com/lindacasas/clon-netflix.git
   ```

2. Navega al directorio del proyecto:

   ```bash
   cd clon-netflix
   ```

3. Abre el archivo `index.html` en tu navegador web preferido.

## USO

1. Navega por las películas disponibles desplazándote por las tarjetas de película.
2. Utiliza la funcionalidad de búsqueda para encontrar películas por título, año, calificación o fecha de estreno.
3. Haz clic en los enlaces de navegación para ordenar las películas por calificación, fecha de estreno o título.
4. Pasa el mouse sobre una tarjeta de película para ver el resumen de la trama.
